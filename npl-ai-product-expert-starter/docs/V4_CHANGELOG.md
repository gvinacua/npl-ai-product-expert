# V4 Changelog

## Added
- Research Scout mode for finding less obvious sources, tools and weak signals.
- Expertise Audit mode for scoring whether outputs are expert-grade or generic.
- Optional web search toggle in the UI.
- Optional deep-model toggle in the UI.
- Backend model selection using OPENAI_MODEL, OPENAI_DEEP_MODEL and OPENAI_SEARCH_MODEL.
- Knowledge packs: Expertise Quality System, Frontier Research Operating Model, Current Source Watchlist, Frontier Signals, Hidden Gems and Tools to Watch.
- One-click buttons: Audit current output and Research upgrade prompt.

## Why this version matters
V3 made the agent useful. V4 makes it testable. The goal is not only to generate better talks, but to create a repeatable process for checking whether a talk is original, specific, fresh, rigorous and worthy of NPL.
