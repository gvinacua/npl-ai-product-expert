# Webinar Delivery Options v1

## Option 1 — Human-led, AI-prepared
The AI expert prepares scripts, slides, examples and Q&A. A human presents. This is safest and immediately usable.

## Option 2 — Synthetic guest speaker clip
The AI expert generates a 3–7 minute voice segment. The host plays it inside the webinar as a prepared guest contribution. Good for demos.

## Option 3 — Operator-controlled live Q&A
The moderator asks questions. A human operator inputs the question into the app, reviews the answer, and either reads it or plays a short voice block. This is the best near-term live mode.

## Option 4 — Voice agent in a parallel browser/device
Use Vapi or OpenAI Realtime to run a voice agent in a browser/device that joins the meeting audio. Requires careful setup, disclosure, and human override.

## Option 5 — Full virtual attendee / avatar
Later stage. Requires Zoom/Teams integration or a meeting bot, identity/disclosure, latency handling, interruption management, and strong safety controls.

## Recommended next step
Build Option 3 first: an operator cockpit with short answer generation, approval, and voice block playback.
