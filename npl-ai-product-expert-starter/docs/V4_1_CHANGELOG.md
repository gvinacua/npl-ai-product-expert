# V4.1 Changelog — Concise Expert Persona

Changes:
- Added concise-by-default interaction discipline.
- Updated NPL AI Product Expert voice to avoid long, generic responses.
- Sparring mode now defaults to one sharp thesis, three bullets and one recommendation.
- Panel mode defaults to 60-90 second answers.
- Lesson mode keeps spoken delivery blocks shorter and more voice-ready.
- Expertise Audit is stricter but shorter by default.
- Added per-mode output token budgets to reduce overlong answers.

Purpose:
The expert should say only what is needed to be clear, useful and sharp.
