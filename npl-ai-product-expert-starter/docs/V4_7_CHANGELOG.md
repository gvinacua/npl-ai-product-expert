# v4.7 — Cost Proxy Display

## What changed

- Added rough euro cost proxy in the UI.
- Added text-generation cost estimate by mode/model toggle.
- Added ElevenLabs audio cost proxy by character count.
- Added per-block audio proxy next to each voice block.
- Added clearer disclaimer: estimates are for decision-making, not billing.
- Kept the previous v4.6 block voice UX.

## Why

The tool needs to be affordable and operationally understandable. Tokens and credits are not intuitive for daily use, so v4.7 introduces approximate decision-useful cost bands.

## Notes

Pricing assumptions are rough and should be updated as provider pricing changes.
